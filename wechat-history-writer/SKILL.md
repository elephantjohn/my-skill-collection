# 微信公众号历史趣闻写作 Skill

## 风格定位

**学习对象：**
- 当年明月（《明朝那些事儿》）- 幽默风趣讲历史
- 马伯庸（历史悬疑）- 细节考据 + 故事性
- 六神磊磊（金庸 + 历史）- 有梗有料

**核心原则（必须遵守）：**
1. **史实第一，幽默第二** - 幽默建立在史实基础上
2. **史料来源，必须标注** - 重要观点注明出处
3. **禁止杜撰，严禁编造** - 不编造对话、细节
4. **区分史实，说明推测** - 史实与推测要明确区分

**核心特点：**
- ✅ 有趣：幽默风趣，不枯燥
- ✅ 真实：尊重史实，有据可查
- ✅ 通俗：通俗易懂，不卖弄
- ✅ 有料：有梗有料，引发思考
- ✅ 负责：对读者负责，经得起考证

## 标题风格

### 1. 悬念式
- "你可能不知道，XXX 其实..."
- "历史上最 XXX 的皇帝，竟然是..."
- "XXX 的背后，藏着一个秘密"

### 2. 反差式
- "被误解千年的 XXX"
- "XXX 的另一面：和你想的不一样"
- "正史里的 XXX，颠覆你的认知"

### 3. 真相式
- "历史上的 XXX：真相是..."
- "XXX 的真实面貌：史料里的记载"
- "揭开 XXX 的历史真相"

### 4. 趣味式
- "如果 XXX 有朋友圈"
- "XXX 的日常：比现代人还..."
- "XXX 的 XXX：古代人的智慧"

## 文章结构

```
【开头】悬念引入（100-150 字）
- 颠覆认知的观点
- 引发好奇心的问题
- 简洁有力的金句

【正文】4-5 个部分（每部分 400-500 字）
1. 历史背景介绍（简洁明了）
2. 有趣的故事细节（生动具体）
3. 历史真相揭秘（有据可查）
4. 人物性格分析（立体饱满）
5. 现代启示思考（引发共鸣）

【结尾】历史智慧（150-200 字）
- 总结历史教训
- 联系现实生活
- 留下思考空间
```

## 语言特色

### ✅ 推荐的表达
- "你可能不知道..."
- "有趣的是..."
- "史料记载..."
- "说白了就是..."
- "用现在的话说..."
- "这操作，绝了..."

### ❌ 避免的表达
- 过度煽情（温情风格）
- 空洞抒情（没有史料）
- 说教口吻（居高临下）
- 戏说过度（违背史实）

## 选题方向

### 1. 历史人物
- 帝王将相的另一面
- 文人墨客的趣事
- 被误解的历史人物

### 2. 历史事件
- 改变历史的关键时刻
- 历史事件的幕后
- 被忽略的历史细节

### 3. 古代生活
- 古人的日常生活
- 古代的衣食住行
- 古人的娱乐方式

### 4. 历史冷知识
- 颠覆认知的真相
- 有趣的历史巧合
- 历史上的今天

## 史料来源

**推荐来源（按优先级）：**

1. **正史（最高优先级）**
   - 二十四史（《史记》《汉书》《三国志》等）
   - 《资治通鉴》
   - 各朝实录

2. **编年体、纪事本末**
   - 《左传》《国语》《战国策》
   - 各朝纪事本末

3. **笔记、野史（需谨慎）**
   - 名人笔记（可参考，需印证）
   - 野史传说（注明"据 XX 记载"）

4. **考古发现**
   - 出土文物
   - 考古报告

**使用原则（必须遵守）：**
1. ✅ 优先使用正史记载
2. ✅ 多方印证（孤证不立）
3. ✅ 注明史料来源（《XXX》载）
4. ✅ 区分史实与推测
5. ✅ 禁止使用"据说"、"相传"等无来源表述
6. ✅ 不编造对话、不编造细节
7. ✅ 对读者负责，经得起考证

**标注格式示例：**
- ✅ "《史记·项羽本纪》载：'项籍者，下相人也'"
- ✅ "据《资治通鉴》记载，此事发生在..."
- ✅ "《三国志》中有这样的描述：..."
- ✅ "考古发现表明..."
- ❌ "据说项羽..."（禁止）
- ❌ "相传他..."（禁止）
- ❌ "有人说这是..."（禁止）

## 微信公众号排版规范（重要！）

### 核心原则
1. **段落间距** - 两段之间两个换行 `<br><br>`
2. **标题层级** - 故事标题 26px，章节标题 28px
3. **颜色搭配** - 红色主标题，蓝色引用，深蓝正文
4. **样式统一** - 用 class 不用内联样式

### 标题格式
- **故事小标题**：`<h2 class="story-title">👑 故事 X：标题</h2>`
  - 26px 红色 + 渐变背景 + 左边框
- **章节标题**：`<h2 class="section-title">【标题】</h2>`
  - 28px 白色 + 紫色渐变背景 + 居中

### 正文格式
- **段落**：`<p>内容</p>` + `<br><br>` 分隔
- **重点**：`<strong>文字</strong>`（20px 红色加粗）
- **启示**：`<div class="lesson">内容</div>`（黄色卡片）
- **列表**：红色大圆点 ●（22px）

### 禁止事项
- ❌ 正文中出现 HTML 代码
- ❌ 直接嵌入 CSS 样式
- ❌ 使用星号 `*`
- ❌ 使用内联样式（除列表圆点）
