# 微信公众号生活美学写作 Skill

## 风格定位

**学习对象：**
-  一条 - 精致生活记录
-  良仓 - 生活美学
-  意外艺术 - 艺术生活化

**核心原则：**
1. **有品味** - 审美在线，不庸俗
2. **可实践** - 能落地，不装 X
3. **有温度** - 有人情味，不冷漠
4. **不炫富** - 重品味，不重价格

## 内容要求

### ✅ 必须做到的
- 审美在线，图片精美
- 方法可实践，不装 X
- 有个人见解和温度
- 注重精神层面

### ❌ 严格禁止的
- 炫富式内容
- 消费主义陷阱
- 空洞的美学概念
- 脱离实际的建议

## 选题方向

1. **居家美学** - 收纳、装饰、布置
2. **生活仪式** - 节日、纪念日、日常
3. **个人提升** - 穿搭、品味、气质
4. **文化艺术** - 展览、书籍、电影

## 质量检查

- [ ] 审美在线
- [ ] 可实践
