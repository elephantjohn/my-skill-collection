# 微信公众号书评影评写作 Skill

## 风格定位

**学习对象：**
-  豆瓣高分评论 - 深度解读
-  虹膜 - 专业影评
-  单读 - 思想深度

**核心原则：**
1. **有观点** - 不人云亦云
2. **有深度** - 不止于表面
3. **不剧透** - 关键情节保密
4. **有延伸** - 联系现实生活

## 内容要求

### ✅ 必须做到的
- 有独特观点和视角
- 深度解读作品内涵
- 关键情节不剧透
- 联系现实生活

### ❌ 严格禁止的
- 剧情复述式评论
- 无观点的流水账
- 恶意剧透
- 人身攻击

## 文章结构

```
【开头】作品引入
- 作品基本信息
- 观看/阅读体验
- 核心观点

【正文】深度解读
1. 主题分析
2. 人物解读
3. 艺术特色
4. 社会意义

【结尾】延伸思考
- 现实启示
- 推荐人群
- 评分
```

## 选题方向

1. **新书速递** - 新书推荐、解读
2. **经典重读** - 经典作品新解
3. **热映电影** - 新片评论
4. **佳片重温** - 老片新看

## 质量检查

- [ ] 有独特观点
- [ ] 无关键剧透
- [ ] 有深度解读
- [ ] 联系现实
