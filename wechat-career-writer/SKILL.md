# 微信公众号职场成长写作 Skill

## 风格定位

**学习对象：**
-  LinkedIn 职场 - 实用干货
-  得到 App - 系统方法论
-  古典 - 成长类写作

**核心原则：**
1. **实用第一** - 方法可执行，不空谈
2. **案例支撑** - 有真实案例，不空洞
3. **数据说话** - 有调研数据，不臆测
4. **避免鸡汤** - 不灌鸡汤，给方法

## 内容要求

### ✅ 必须做到的
- 提供具体可执行的方法
- 有真实案例或数据支撑
- 指出适用场景和局限性
- 给出行动建议

### ❌ 严格禁止的
- 空洞的鸡汤文
- 无来源的"成功学"
- 绝对化的建议
- 脱离实际的方法

## 文章结构

```
【开头】痛点引入（100-150 字）
- 职场常见痛点/困惑
- 引发共鸣的场景
- 核心观点预告

【正文】4 个部分（每部分 400-500 字）
1. 问题分析（为什么难）
2. 方法论（怎么做）
3. 案例展示（别人怎么做）
4. 行动建议（你该怎么做）

【结尾】总结行动（150-200 字）
- 核心方法总结
- 具体行动步骤
- 鼓励与支持
```

## 选题方向

1. **职场技能** - 沟通、时间管理、汇报
2. **职业发展** - 晋升、跳槽、转行
3. **人际关系** - 上下级、同事、客户
4. **心态成长** - 压力管理、自我认知

## 质量检查

- [ ] 方法可执行
- [ ] 有案例支撑
- [ ] 无空洞鸡汤
- [ ] 指出局限性
