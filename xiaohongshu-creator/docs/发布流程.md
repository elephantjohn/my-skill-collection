# 📱 小红书发布流程说明

## ⚠️ 重要说明

**小红书没有官方发布 API**，所有内容生成后需要**手动发布**到小红书 APP。

---

## 🔄 完整工作流程

```
定时任务触发 (每 10 小时)
       ↓
1. Gemini 3 Pro 生成文案
       ↓
2. 百度智能审核 (分段定位 + 局部修改)
       ↓
3. 生成 3 个爆款标题 + 8 个标签
       ↓
4. Veo 3.1 生成视频 (9:16, 720p, 8s)
       ↓
5. 保存到本地 (JSON + TXT)
       ↓
6. 发送到 Telegram (给你)
       ↓
7. 你手动复制并发布到小红书 APP ✅
```

---

## 📋 发布步骤

### 第 1 步：接收内容
定时任务完成后，你会在 Telegram 收到：
- 📝 3 个爆款标题（选 1 个）
- 📄 完整文案
- 🏷️ 8 个热门标签
- 🎬 视频文件路径

### 第 2 步：准备素材
```bash
# 查看生成的内容
cd /home/ubuntu/.openclaw/workspace-xiang_xiaohongshu/skills/xiaohongshu-creator/output/
ls -la  # 查看最新文件
cat note_*.txt  # 查看文案
```

### 第 3 步：打开小红书 APP
1. 打开手机小红书 APP
2. 点击底部 "+" 发布按钮
3. 选择"上传视频"

### 第 4 步：上传视频
```bash
# 视频文件位置
/home/ubuntu/.openclaw/workspace-xiang_xiaohongshu/skills/xiaohongshu-creator/videos/
# 或
/home/ubuntu/.openclaw/workspace-xiang_xiaohongshu/skills/gemini-video/
```

**上传方法**：
- 方法 A：用数据线传到手机
- 方法 B：用微信/QQ 发送到手机
- 方法 C：上传到云盘后下载

### 第 5 步：填写内容
1. **标题**：从 3 个选项中选择 1 个
2. **文案**：复制完整文案
3. **标签**：复制 8 个标签（自动添加#）
4. **封面**：选择视频中最精彩的一帧

### 第 6 步：发布
1. 选择分类（建议"日常"或"解压"）
2. 设置可见范围（公开）
3. 点击"发布笔记"

## 💡 优化建议

### 内容优化
- ✅ 文案保持 300-500 字

- ✅ 结尾引导互动（评论/点赞）

- ✅ 前 3 秒必须吸引人
- ✅ 时长 8-15 秒最佳
- ✅ 高清 720p 以上

### 互动优化
- ✅ 发布后 1 小时内回复评论
- ✅ 主动在评论区提问

## ❓ 常见问题

### Q: 为什么不能自动发布？
A: 小红书没有开放 API，自动发布需要 RPA 模拟浏览器，有封号风险。

### Q: 视频怎么传到手机？
A: 推荐用微信文件传输助手，或上传到百度网盘后手机下载。

### Q: 可以一次发多篇吗？
A: 建议每天 1-2 篇，过多可能被限流。

### Q: 标题可以修改吗？
A: 当然可以！生成的标题仅供参考，你可以根据经验调整。
